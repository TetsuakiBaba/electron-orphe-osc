# electron-orphe-osc
makes [ORPHE-CORE.js](https://orphe-oss.github.io/ORPHE-CORE.js/) enable to use OSC communication with Electron.

## Getting Started
```
  > npm install electron
  > npm install osc-js
  > npm start
```
have a fun!
